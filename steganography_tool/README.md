# Steganography Tool Helpers

This folder originally contained simple proof-of-concept scripts for hiding
text, images and executables inside other files. Those helpers were incomplete
and are no longer shipped. The main CLI under `main.py` and the plugin system in
`stego_plugins/` provide the recommended functionality.

```
python main.py
```

Use the CLI menus to hide or extract data from images, PDF files and more.
